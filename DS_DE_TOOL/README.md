# Data Engineering Toolkit

Multi-phase data processing pipeline combining streaming, caching, and API serving.

## Tech Stack
- **Streaming:** Apache Kafka (confluent_kafka)
- **Caching:** Redis
- **API:** FastAPI
- **Language:** Python

## What It Does
- Phase 1-3 data ingestion and transformation pipeline
- Real-time data streaming with Kafka producer/consumer
- Redis-backed caching layer for fast lookups
- FastAPI endpoints for serving processed data
