# Data Science Projects

Academic coursework from Bachelor's and Master's programs at Chulalongkorn University.

## Highlights

### Master's - Final Project: Data Engineering & NLP Pipeline
End-to-end academic paper analysis system:
- **Data Collection:** Scopus API + pybliometrics for paper retrieval (20K+ papers)
- **Pipeline:** Kafka streaming, Redis caching, FastAPI serving
- **ML Model:** BERT (Hugging Face) multi-label classifier for 18 engineering categories
- **Tracking:** Weights & Biases (W&B) experiment logging
- **Tools:** PySpark, NLTK, Scikit-learn

### Master's - Midterm: Scopus Dataset Analysis
Text classification on academic paper metadata using transformer models.
