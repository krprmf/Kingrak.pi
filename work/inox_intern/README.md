# INOX Internship - Video Content Analyzer

Internship project at INOX (Innovative Extremist Co., Ltd.) - Deeptech Team.

## What It Does
LLM-powered video transcript (VTT) analyzer that extracts structured information from video content.

## Features
- Parses VTT subtitle files (Thai & English)
- Engineered prompts for structured JSON extraction including:
  - Timestamps, segment summaries, content tags
  - Topics, keywords, emotional tone
  - Highlighted moments & informational value scores
- Configurable segmentation resolution (low/medium/high)
- Multi-language support (Thai/English output)

## Tech
- **Core:** Generative AI, LLM Prompt Engineering
- **Output:** Structured JSON schema
- **Languages:** Thai, English
